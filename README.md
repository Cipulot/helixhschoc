# Helix HS Choc

![Helix HS Choc](/Doc/images/Helix_hs_choc.jpg)

Hotswap Choc version of the Helix, a compact split ortholinear keyboard.

Original Helix keyboard was designed by [Makoto Kurauchi](https://github.com/MakotoKurauchi).
